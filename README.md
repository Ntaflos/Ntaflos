A real-world business email and newsletter infrastructure case study using **AWS SES**, **Google Workspace**, **DNS records**, **SMTP**, **SPF**, **DKIM**, **DMARC**, monitoring and email deliverability practices.

This project helped me understand how email authentication, domain trust, cloud configuration and monitoring support secure business communication.

### 🎓 [Spatial & Spatio-Textual Data Visualization Platform](https://github.com/Ntaflos/diploma_thesis_master)

Academic software engineering project developed as part of my diploma thesis at the University of the Aegean.

Built with **Java**, **Spring Boot**, **Spring MVC**, **HTML**, **CSS**, **JavaScript**, **Leaflet**, **Hibernate** and **GeoJSON**.

The project allowed users to import spatial data from multiple sources, execute configurable queries, and visualize results on an interactive map.
